## Running the Application

```bash
# Install Ionic CLI globally
npm install -g @ionic/cli
```

```bash
# Install project dependencies
npm install
```

```bash
# Start the development server
ionic serve
```

## Note on Dependencies
This project was developed in 2023. Some dependencies are outdated or have known vulnerabilities; this project is intended for learning and portfolio purposes only, not for production use.
